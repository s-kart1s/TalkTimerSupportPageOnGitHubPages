# TalkTimer Website

このディレクトリには、TalkTimerアプリのGitHub Pagesサイトが含まれています。

## 公開URL

GitHub Pagesの設定後、以下のURLでアクセスできます：

```
https://[あなたのGitHubユーザー名].github.io/TalkTimer/
```

## ファイル構成

- `index.html` - メインページ（アプリの紹介）
- `privacy-policy.html` - プライバシーポリシー
- `support.html` - サポート・FAQ

## GitHub Pagesの設定方法

1. GitHubリポジトリにこのディレクトリをプッシュ
2. リポジトリの Settings > Pages に移動
3. Source で `main` ブランチの `/docs` フォルダを選択
4. Save をクリック

数分後、サイトが公開されます。

## ローカルでのプレビュー

ローカルで確認する場合は、`docs/index.html`をブラウザで開くだけです。

## カスタマイズ

- GitHubのユーザー名やリポジトリ名を実際のものに変更してください
- 連絡先情報を追加する場合は、各HTMLファイルを編集してください
